## ORDER BY

 - order by 뒤에 붙인다. null을 먼저 정렬할지, 나중에 정렬할지 결정한다. 

```sql
select * from emp 
order by comm desc nulls first
```

## like 에 % 말고 _도 사용하자
```sql
--SALES뒤에 xxx 세글자가 붙는 패턴
select * from emp
where job like 'SALES___'

--ES뒤에 xxx 세글자가 붙고 앞에는 글자수 상관없이 xx..로 시작
select * from emp
where job like '%ES___'
```
